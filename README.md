# 정찬호 개발 포트폴리오

수업 학습 과정과 프로젝트 경험을 함께 정리한 정적 포트폴리오입니다.

## 구성

- 전체 수업 커리큘럼: Web/HTML부터 AWS 배포까지 14개 영역
- BIDEO: 작품·예술관·경매·결제·S3·AI 기능
- Clausion: AI 기반 온라인 교육 플랫폼
- GGShop: Spring/MyBatis 기반 웹 서비스
