Features

핵심 기능

🎥

실시간 화상 강의

LiveKit 기반 WebRTC로 지연 없는 화상 수업을 구현했습니다. WebSocket 채팅으로 강의 중 실시간 소통이 가능하며, 강의 녹화 기능도 지원합니다.

🤖

AI 학습 챗봇

OpenAI API 기반 학습 도우미(ChatbotAi.java)로 학생이 언제든 질문할 수 있는 24/7 AI 튜터를 구현했습니다. 강의 컨텍스트를 유지하며 단계별 힌트를 제공합니다.

💻

AI 코드 리뷰

학생이 제출한 코드를 AI가 자동 분석합니다(CodeReviewAi.java). 문법 오류부터 최적화 제안, 코드 스타일 피드백까지 즉각적인 리뷰로 학습 속도를 높입니다.

🎮

게임화 시스템

XP(경험치), 레벨, 뱃지 시스템으로 학습 동기를 부여합니다(GamificationService.java). 출석 연속 기록, 과제 완료 보상 등 다양한 게임 요소로 학습 지속률을 향상시켰습니다.

🧬

Digital Twin 학습 분석

학생의 학습 패턴을 분석해 가상 분신 AI 모델을 구축합니다(TwinInferenceEngine.java). 취약 개념 식별, 최적 학습 경로 추천, 개인화 커리큘럼을 자동 생성합니다.

📅

Spaced Repetition 복습 스케줄러

에빙하우스 망각 곡선 알고리즘 기반 최적 복습 일정을 자동 생성합니다(ReviewScheduler.java). 개인별 기억 강도를 추적해 가장 효율적인 복습 타이밍에 알림을 전송합니다.

👥

스터디 그룹 매칭

학습 목표, 관심 분야, 수준, 선호 시간대 등 다차원 데이터를 분석해 최적의 스터디 파트너 및 그룹을 자동으로 추천합니다. 그룹 내 코드 공유와 실시간 협업도 지원합니다.

AI Components

AI 컴포넌트 상세

ConsultationCopilot

강사 상담 지원 AI

강사가 학생을 상담할 때 AI가 해당 학생의 학습 이력, 취약점, 최근 행동 패턴을 실시간으로 분석해 상담 인사이트를 제공합니다. Anthropic SDK 기반으로 정확한 자연어 이해를 구현했습니다.

TwinInferenceEngine

개인화 학습 분석

학생의 모든 학습 데이터를 수집해 개인화된 AI 모델을 구축합니다. 학습 속도, 오답 패턴, 집중도 변화를 추론하여 맞춤형 학습 경로와 예측 기반 조기 개입을 가능하게 합니다.

CodeReviewAi

코드 품질 자동 평가

OpenAI API를 활용해 제출 코드의 정확성, 효율성, 가독성, 보안 취약점을 다각도로 평가합니다. 단순 오류 지적을 넘어 더 나은 코드 작성 방법까지 교육적 피드백을 제공합니다.

ReviewScheduler

에빙하우스 기반 복습 최적화

망각 곡선 이론을 알고리즘으로 구현해 각 개념의 기억 유지율을 예측합니다. 기억이 약해지는 최적의 타이밍에 복습 알림을 전송해 장기 기억 형성 효율을 극대화합니다.
