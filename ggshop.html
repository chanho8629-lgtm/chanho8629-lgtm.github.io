Features

핵심 기능

📊

실시간 대시보드

ag-Grid Community Edition으로 1,000개 이상의 충전소 현황을 실시간으로 표시합니다. 셀 렌더러로 상태별 색상 구분, 인라인 편집, 정렬·필터를 모두 구현했습니다.

📈

멀티 차트 시각화

Chart.js를 활용해 일별 사용량 라인 차트, 지역별 분포 바 차트, 충전 타입별 파이 차트를 구현했습니다. 데이터 변경 시 애니메이션과 함께 실시간으로 업데이트됩니다.

🗺️

충전소 CRUD 관리

충전소 등록·수정·삭제 전체 기능을 구현했습니다. 시·도·구 드롭다운 연동 지역 필터, 충전 타입·운영 상태 검색, 키워드 통합 검색으로 원하는 충전소를 빠르게 찾습니다.

👤

다중 권한 계정 시스템

관리자(전체 기능) / 운영자(담당 지역 충전소 관리) / 일반 사용자(조회 전용) 3단계 권한을 Spring Security 인터셉터로 구현했습니다. 권한별 메뉴 표시도 동적으로 처리합니다.

📋

보고서 출력 / 엑셀 다운로드

기간별 충전 사용량 집계 보고서를 화면에서 직접 확인하고 Apache POI로 엑셀 파일로 다운로드할 수 있습니다. 지역별·타입별 그룹핑 집계도 지원합니다.

Retrospective

이 프로젝트에서 배운 것

핵심 학습 포인트

  • SSR vs CSR의 실질적 차이 — Thymeleaf로 완전한 서버사이드 렌더링을 구현하며, SEO·초기 로딩 속도와 개발 복잡도 사이의 트레이드오프를 직접 체험했습니다.
  • ag-Grid 셀 렌더러 / 에디터 커스터마이징 — 기본 그리드를 넘어 커스텀 렌더러로 상태 뱃지, 인라인 액션 버튼, 조건부 스타일링을 구현하며 라이브러리 내부 구조를 이해했습니다.
  • Chart.js 실시간 업데이트 패턴 — setInterval 폴링 + chart.update() 패턴으로 데이터를 30초마다 자동 갱신하며, 애니메이션 프레임 최적화도 학습했습니다.
  • Spring MVC 인터셉터 인증 흐름 — HandlerInterceptor로 요청 전 인증·인가를 처리하며, Filter와 Interceptor의 역할 차이와 적절한 사용 시점을 이해했습니다.
  • 대용량 데이터 페이지네이션 전략 — MyBatis의 Page 쿼리 최적화로 10만 건 이상 데이터에서도 빠른 응답 시간을 유지하는 SQL 튜닝을 경험했습니다.
